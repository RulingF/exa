##################
Installation
##################
| |conda|
| |pypi|
Exa is available through `anaconda`_,

.. code-block:: bash

    conda install exa

or `pypi`_.

.. code-block:: bash

    pip install exa

###################
Getting Started
###################
| |docs|
| |gitter|
Documentation can be built using `sphinx`_:

.. code-block:: bash

    cd doc
    make html    # or .\make.bat html
